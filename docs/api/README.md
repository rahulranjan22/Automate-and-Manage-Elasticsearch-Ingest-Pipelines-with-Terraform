## API Reference

This reference documents the public interfaces exposed by this repository. Since the repository currently contains only a top-level README, the following sections provide a template you can populate as code and modules are added.

Contents:
- Terraform modules
- Input variables
- Output values
- Makefile/CLI commands
- CI workflows

### Terraform Modules

For each module, create a file under `docs/api/modules/<module-name>.md` with:
- Description
- Inputs (variables)
- Outputs
- Resources/Data sources
- Usage example

### CLI/Make Targets

If you provide helper scripts or Make targets, document them under `docs/api/cli.md` with usage, flags, and examples.

### CI/CD

If you ship reusable workflows, document inputs and outputs under `docs/api/ci.md`.
